CRISIS MANA - A character cross inpsired card game

 Crisis mana is a board game, created by the B.A.D. ITB Team (Billinguis Amicus Divisio). It takes great inspiration from a previous work "Character cross", which is a card game revolving around characters using skills controlled by dice and cards to combat enemies. Crisis mana aims to reduce the RNG (luck) factor of character skills by allowing skills to be used without needing to rely on luck, whilst still introducing RNG gimmicks to keep the playing field intense.
